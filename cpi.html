How To Use Application

Initial Set-up

Admin has to login with the Admin User ID and password.

Admin will be able to add attributes, select attributes and view reports

Everyday use

Membership Data Entry:
a. Enter Data: Menu -> Membership Data Entry -> Enter Data
Before entering the data, look up for the Company name and year that is auto genegrated.
Select the month range for which the data is to entered.
Provide the data for the KPI displayed.
Save the data for aggregate comparision
b. Edit Data:Menu -> Membership Data Entry -> Edit Data
Edit data page, will allow you to view the previously entered data
Any changes can be updated to data retrieved.
Terms and Conditions:
This option can be used to view terms and conditions to use the application.

Add Attributes:
a. Add KPI: Menu -> Add Attributes -> Add KPI
Admin can add KPI as requirement arise.
b. Select KPI:Menu -> Add Attributes -> Select KPI
A table containing all the KPI will appear, allowing the Admin to select the required KPI's for data collection
c. Add Month Range: Menu -> Add Attributes -> Add Month Range
Month range can be selected depending on the month range the Admin wants to collect data for.
d. Add Business Type:Menu -> Add Attributes -> Add Business Type
New Business Type can be added by the Admin using this option.
User:
Show the users details, including company name and the business type.

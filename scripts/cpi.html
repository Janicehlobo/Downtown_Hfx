How To Use Application

Initial Set-up

Each user should be assigned to a co-operative to use the application. You can use the “Profile” page to check whether your co-operative has been registered in the application or not.

Profile:
a. Add Co-operative: Menu -> Profile -> Add Your Coop
To add your co-operative, use “Add Your Coop” on the “Profile Page”. A pop-up reminds you before adding your co-operative to the application to check registered co-operatives to avoid duplicate registration.
b. Assign user to Co-operative:
If you add a co-operative to the application, you will automatically be assigned to it. Otherwise, ask the application administrator to assign you to a co-operative.
c. Edit user information / co-operative information:
Use the edit icon in the “Action” column to edit information. The application will open an edit page, you can then make changes and use the “Save changes” button to save and go back to the profile page. Use the “Cancel” button to discard changes and go back to the profile page.

Everyday use

CPI Data Entry:
a. Enter Data: Menu -> CPI Data Entry -> Enter Data
To enter data, first choose “Co-operative Name” (the dropdown list shows the name of the co-operative that you are assigned to, however if it is empty it means that you are not assigned to any co-operative, therefore you should ask an administrator to assign you to a co-operative), then select “Year”, “Fiscal Year”, “Asset Size”, and “Revenue Size” and enter data for each KPI. Use the “Save” button to save data or the “Cancel” button to discard changes and go back to previous page.
b. Edit Data: Menu -> CPI Data Entry -> Edit Data
To edit data, first choose “Co-operative Name” (the dropdown list show the name of the co-operative that you are assigned to, however if it is empty it means that you are not assigned to any co-operative, therefore you should ask an administrator to assign you to a co-operative), then select “Year” (the dropdown list contains related years for the co-operative), “Asset Size” and “Revenue Size” which have been previously entered will be shown in their fields and “Fiscal year” will be shown in an additional pop-up window. Choose “Fiscal Year” and enter data for all the KPIs; if you leave any of them empty, zero will be inserted into the database for those fields. Use the “Save changes” button to save data or the “Cancel” button to discard changes and go back to previous page.

KPI Definition:
This option can be used to view the full definition of each KPI.

Terms and Conditions:
This option can be used to view terms and conditions to use the application.

Report:
This option can be used to generate reports based on “Co-operative”, “Industry”, “Province”, and “Country”. You also have options to download the report as PDF or CSV files, or send the PDF format to your registered email. In each step you can use the “Cancel” button to go back to the main report page. Please note that you can generate reports based on “Industry” and “Province” for the industry and the province related to your co-operative if there are more than two registered co-operatives in the same field.

How To Use Application

Initial Set-up

To use the application, you first need to set up the basic attributes for co-ops and then define the KPIs. You can revise the attributes and KPIs based on your needs at any time.

Add Attributes
This option can be used to add, deactivate and activate attributes.
a. Add New Attributes: Menu -> Add Attributes
You can use this page to add “Industry” and “Type” that will be used to define co-operatives, add “Category” that will be used to define KPIs, and add “Asset Size” and “Revenue Size” that will be use to enter data.
b. Activate/ Deactivate Attributes: Menu -> Add Attributes
Use the edit icon in the “Action” column on the “Add Attribute Page”, a pop-up window will then open. To deactivate the selected Attributes, select “Inactive” in the first dropdown list. You can activate an Attribute by changing its status to “Active”.

KPI
This option can be used to add, edit, deactivate and activate KPIs.
a. Add new KPI: Menu -> KPI -> Add New KPI
Use the “Add New KPI” button on the “KPIs Page”. The application will open a new page to define a new KPI (please note that all fields are mandatory to fill), then use the “Save” button to add the KPI. You could also use the “Cancel” button to go back to the “KPIs Page” without saving a new KPI.
b. Edit KPI: Menu -> KPI
Use the edit icon in the “Action” column on the “KPIs Page”. The application will open a new page to edit the selected KPI.
c. Activate/ Deactivate KPI: Menu -> KPI
Use the edit icon in the “Action” column on the “KPIs Page”. The application will open a new page. To deactivate the selected KPI, select “Inactive” in the “Status” dropdown list. You can activate a KPI by changing its status to “Active”.

Everyday use

Cooperatives
This option can be used to add, edit, deactivate and activate co-operatives.

User
This option can be used to add, edit, deactivate and activate users, as well as upgrade a user's role to "admin".

Comment
This option can be used to leave a comment for users. The comment will be posted on the user’s home page.

CPI Data Entry
This option can be used to enter or edit data.
a. Enter Data: Menu -> CPI Data Entry -> Enter Data
To enter data, first choose “Co-operative Name” (the dropdown list contains all active registered co-operatives), then select “Year”, “Fiscal Year”, “Asset Size”, and “Revenue Size”, and enter data for each KPI. Use the “Save” button to save data or the “Cancel” button to discard changes and go back to previous page.
b. Edit Data: Menu -> CPI Data Entry -> Edit Data
To edit data, first choose “Co-operative Name” (the dropdown list contains active registered co-operatives, which have previously entered data in the database), then select “Year” (the dropdown list contains related years for the selected co-operative). “Asset Size” and “Revenue Size” which have been previously entered will be shown in their fields and “Fiscal year” will be shown in an additional pop-up window. Choose “Fiscal Year” and enter data for all the KPIs; if you leave any of them empty, zero will be inserted into the database for those fields. Use the “Save changes” button to save data or the “Cancel” button to discard changes and go back to previous page.

Report
This option can be used to generate reports based on “Co-operative”, “Industry”, “Province”, and “Country”. You also have options to download the report as PDF or CSV files, or send the PDF format to your registered email. In each step, you can use the “Cancel” button to go back to the main report page.
